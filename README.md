## Overview

A simple, in-progress, text-based RPG concept using C++ and Qt.

## Getting Started

1. Open the project `CMakeLists.txt` file within Qt Creator to auto-load and configure the project.
2. Build -> Build Project "DSTRPG"
3. Ctrl + R to run


### Prerequisites

- Qt Creator
- Qt 6.7
- CMake
